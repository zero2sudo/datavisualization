# DataViz Studio

A drag-and-drop data visualization tool built with React, TypeScript, and Vega-Lite.

## About This Project

This repository is a demonstration of [Claude Code](https://clau.de/zero2sudo) and Claude Opus 4.5, created by [zero2sudo](https://instagram.com/zero2sudo) in partnership with Anthropic. Watch the original video: [instagram.com/p/DSgWeRhAb1j](https://www.instagram.com/p/DSgWeRhAb1j/)

## Features

- Drag-and-drop interface for building charts
- Automatic field type detection (quantitative, nominal, temporal)
- Real-time Vega-Lite chart rendering
- Support for X, Y, Color, and Size encodings
